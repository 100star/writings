http://cappuccino.org

280north 에서 개발하고 있습니다.
http://280north.com/

Cappuccino 는 Objective-C 와 유사한 Objective-J 로 만들어졌습니다.
Cappuccino 사용자들도 Objective-J 를 사용해야 합니다.
Objective-J to JavaScript 컴파일러도 280north 에서 제공합니다.

Cappuccino 개발 도구들은 JavaScript 로 만들어졌기 때문에 자바스크립트 실행환경이 필요합니다.
280north 에서 narwhal 이란 자바스크립트 환경을 제공합니다.
narwhal 에서 사용하는 패키지 메니저 명령은 tusk 입니다.
http://narwhaljs.org/

Cappuccino 설치

아래 페이지에 가시면 Starter Package 라는 것이 있습니다.
http://cappuccino.org/download/

압축파일 안에 있는 NewApplication 은 추가 도구 없이 카푸치노 어플리케이션을 만들어 보 수 있는 템플릿입니다.
폴더의 index.html 을 브라우저도 불러 보시면 Hello World 가 실행됩니다.
단, 현재 크롬은 보안 정책상 file:/// 에서 카푸치노를 실행할 수 없으니 다른 브라우저를 사용하셔야 합니다.

카푸치노에 손을 담그기 위해 관련 도구들을 설치하시려면 bootstrap.sh 을 실행합니다.
narwhal, objective-j, jake, 등 280north 에서 제작한 자바스크립트 환경과 개발 툴들이 설치됩니다.

기본으로 따라오는 rhino 는 자바로 만들어진 자바스크립트 엔진이기 때문에 기동이 느립니다.
맥 환경이라면 설치 과정중 JavaScriptCore 어뎁터를 설치할 것인지 묻는데 가능하면 이것을 쓰도록 합니다.
JavaScriptCore 는 OS X 에 기본 설치되어 있는 자바스크립트 엔진입니다.

Cappuccino 어플리케이션 생성

오랜 bootstrap.sh 설치 과정이 끝나고 적당한 디렉토리에서 아래 명령을 치면 어플리케이션이 생성됩니다.
capp gen MyApplication

MyApplication 폴더 안의 index.html 을 브라우저에서 실행합니다.
물론 이때도 크롬은 안 됩니다.
