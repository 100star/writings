애플 디벨로퍼 사이트가서 iOS SDK 를 받아 설치하려고 했더니 여기 또 Xcode 가 들어있네요. =,=
iOS SDK 만 분리해서 다운로드할 수 없게 되어있습니다.
전 레오파드 디스크에서 Xcode 를 왜 설치한 것일까요. =,=

해외 라인 안 좋은 케이블 회사 인터넷 쓰는지라 3.5 기가 다운에 23 시간 뜨네요.
IDC 에 들어있는 우분투 서버에서 30 분 만에 다운로드 후 다시 집으로 30 분 다운로드로 해결.

설치한 Xcode 는 3.X 입니다.
11 월 현재 Xcode 4 Preview 5 는 시뮬레이터가 잘 죽어서 사용을 포기했습니다.

소문대로 Xcode 에디터는 Intellij IDEA 보다 편집 기능이 많이 약하군요.
Code Completion 후보 보는 숏컷이 Opt-ESC 네요. =o=
바꾸면 되겠지만 하루에도 수백 번 사용할 저 키를 매번 어떻게 누르라는 것일까요?
키 누르는 것도 귀찮아서 IDEA 에서는 자동으로 펼쳐지게 해 놓고 쓰고 있는데.

편집도구 불편한 것은 익숙해 지면 나름 극복이 될 터인데,
그보단 Objective-C 의 괴상한 문법이 제일 큰 장애물입니다.

C++ 과 다르게 Dynamic Typing 언어였군요.
C 문화이니 코딩시 타입을 다 적어주면 코드 컴플리션 문제는 없을 것 같고,
런타임 바인딩이라 C++ 처럼 오브젝트 라이브러리 업데이트되면 황당해지는 상황도 없을 것 같긴 합니다.

문제는 과도한 괄호, 대괄호 사용 문제.
코딩을 하다 보면 줄 앞으로 돌아가서 대괄호를 추가해야할 상황이 많이 발생하지 않을까 싶은데,
이 문제는 어떻게 극복하고 있는지 아직 모르겠습니다.
격어 보면 알겠지만.

프로퍼티 하나 추가하는데도 파일을 3 군데나 돌아다녀야 하는군요. 역시.
